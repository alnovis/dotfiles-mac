# Горячие клавиши Neovim

Конфигурация на базе LazyVim с темой Kanagawa. Все кастомные клавиши дублированы для кириллической раскладки.

> `<leader>` = `Space` (пробел)

## Навигация

| Клавиша | Режим | Описание |
|---------|-------|----------|
| `h/j/k/l` | n | Влево / Вниз / Вверх / Вправо |
| `w` | n | Следующее слово |
| `b` | n | Предыдущее слово |
| `e` | n | Конец слова |
| `0` | n | Начало строки |
| `$` | n | Конец строки |
| `^` | n | Первый непробельный символ |
| `gg` | n | В начало файла |
| `G` | n | В конец файла |
| `{` / `}` | n | Предыдущий / следующий абзац |
| `Ctrl+d` | n | Полстраницы вниз |
| `Ctrl+u` | n | Полстраницы вверх |
| `%` | n | К парной скобке |
| `Ctrl+o` | n | Назад по истории переходов |
| `Ctrl+i` | n | Вперёд по истории переходов |
| `Ctrl+Click` | n | Перейти к определению (LSP) |
| `Ctrl+Alt+Click` | n | Найти использования (LSP) |
| `Ctrl+Alt+Left` | n | Назад |
| `Ctrl+Alt+Right` | n | Вперёд |
| `Cmd+[` / `Cmd+]` | n | Назад / вперёд (macOS) |

## Выделение

| Клавиша | Режим | Описание |
|---------|-------|----------|
| `v` | n | Посимвольное выделение |
| `V` | n | Построчное выделение |
| `Ctrl+v` | n | Блочное выделение |
| `viw` | n | Выделить слово под курсором |
| `vi"` | n | Выделить внутри кавычек |
| `vi(` | n | Выделить внутри скобок |
| `vi{` | n | Выделить внутри фигурных скобок |
| `vit` | n | Выделить внутри тега |
| `vap` | n | Выделить абзац |
| `ggVG` | n | Выделить весь файл |
| `j` / `k` | v | Расширить выделение вниз / вверх |
| `o` | v | Переключить курсор на другой конец выделения |

## Редактирование

| Клавиша | Режим | Описание |
|---------|-------|----------|
| `i` / `a` | n | Вставка перед / после курсора |
| `I` / `A` | n | Вставка в начало / конец строки |
| `o` / `O` | n | Новая строка снизу / сверху |
| `jk` | i | Выход из режима вставки (также `ол`) |
| `Ctrl+s` | n, i | Сохранить файл (также `Ctrl+ы`) |
| `u` | n | Отменить |
| `Ctrl+r` | n | Повторить |
| `.` | n | Повторить последнее действие |
| `dd` | n | Удалить строку |
| `yy` | n | Скопировать строку |
| `p` / `P` | n | Вставить после / перед |
| `yyp` | n | Дублировать строку |
| `yap` + `}p` | n | Дублировать абзац |
| `V` + выделить + `y` + `p` | v | Дублировать блок |
| `cc` | n | Заменить всю строку |
| `ciw` | n | Заменить слово |
| `ci"` | n | Заменить внутри кавычек |
| `>>` / `<<` | n | Увеличить / уменьшить отступ |
| `>` / `<` | v | Отступ выделения |
| `Alt+j` | n, v | Переместить строку(и) вниз |
| `Alt+k` | n, v | Переместить строку(и) вверх |
| `~` | n | Переключить регистр символа |

## AI (CodeCompanion + Ollama)

Модель: `qwen3.5:9b`. Ollama запускается автоматически при первом использовании.

| Клавиша | Кириллица | Режим | Описание |
|---------|-----------|-------|----------|
| `<leader>ac` | `<leader>фс` | n | Открыть AI-чат |
| `<leader>ac` | `<leader>фс` | v | Чат с выделенным кодом |
| `<leader>ag` | `<leader>фп` | n, v | Меню AI-действий (Explain, Refactor, Fix, Tests) |
| `<leader>ai` | `<leader>фш` | n, v | AI inline-промпт |

### Внутри чата

| Клавиша | Описание |
